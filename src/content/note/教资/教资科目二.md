# 一. 教育基础知识
## 1.1 教育的产生于发展
教育来源于  孟子尽心上  得天下英才而教育之
教育的定义： 广义： 学校家庭社会
教育活动的基本要素：教育者， 受教育者，教育影响

教育的属性：
本质属性： 有目的 人 社会活动
社会属性： 社会永历利

教育的起源：
1. 本能生利息
2. 心理仿梦露
3. 米凯爱劳动
4. 叶澜爱交往

教育的历史发展
古代教育：劳教分离有阶级
